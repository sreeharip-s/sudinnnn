Keywords

Reserved words. Cannot be used as variable names.

False None True and as assert break class continue def del elif else except finally for from global if import in is lambda nonlocal not or pass raise return try while with yield

Case Sensitivity Warning

`if` is a keyword. `If` is not. `IF` is not.

If(True): # Syntax Error! print("Hi")

Identifiers

Rules for naming your variables.

The 3 Golden Rules

  • 1 Must start with a Letter (A-Z) or Underscore (_).
  • 2 Can contain Letters, Digits (0-9), or Underscores.
  • 3 Cannot contain Spaces or Special Symbols (like @, $, %).
Valid Name ✅ Invalid Name ❌

Variables

Dynamic typing and assignment.

Dynamic Typing

You don't need to tell Python what type of data a variable holds. It figures it out on its own... Which can be a bother at times.

x = 10 # Int x = "Hello" # Now String

LValue Rule

Left side must be a variable container.

Correct
a = 10
Wrong
10 = a

Input & Output

Talking to the user.

Input

input()

Always returns a String.

age = input("Age: ")

Conversion Required

If you want math, use int().

int(input("Age: "))

Output

print()

Displays data to screen.

Separator

print("A","B", sep="-") # A-B

End Character

print("Hi", end="")
